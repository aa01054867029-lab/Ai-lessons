# Directives

This directory contains SOPs (Standard Operating Procedures) written in Markdown.

Each directive defines:
- **Goal** — what needs to be accomplished
- **Inputs** — what data/context is needed
- **Tools/Scripts** — which execution scripts to call
- **Outputs** — expected deliverables
- **Edge Cases** — known pitfalls and how to handle them

Directives are living documents — they get updated as the system learns.
