# Подключение Notion MCP

## Цель
Добавить официальный MCP-сервер Notion в конфигурацию Antigravity, чтобы агент получил доступ к Notion-инструментам (поиск, чтение, создание страниц и баз данных).

## Входные данные
Нет. Скрипт не требуется — нужно только отредактировать JSON-файл.

## Действия

### 1. Открыть конфиг MCP
Файл находится по пути:
```
~/.gemini/antigravity/mcp_config.json
```

### 2. Добавить Notion-сервер
Добавь в объект `mcpServers` блок `"notion"`:

```json
{
  "mcpServers": {
    "notion": {
      "serverUrl": "https://mcp.notion.com/mcp"
    }
  }
}
```

**Если в конфиге уже есть другие серверы** — добавь `"notion"` рядом с ними, не удаляя существующие.

### 3. Сообщить пользователю
После сохранения конфига скажи пользователю:

> Я добавил Notion MCP-сервер в конфигурацию. Теперь нужно завершить подключение вручную:
> 1. Нажми **"..."** в правом верхнем углу панели чата Antigravity
> 2. Выбери **"Manage MCP Servers"**
> 3. Пройди авторизацию через Notion (OAuth)
> 4. Вставь полученный API-ключ и нажми Submit
>
> После этого я смогу работать с твоим Notion.

## Выходные данные
- Обновлённый файл `~/.gemini/antigravity/mcp_config.json` с блоком Notion

## Крайние случаи
- **Файл не существует**: Создай файл `~/.gemini/antigravity/mcp_config.json` с полным конфигом
- **Конфиг содержит другие серверы**: Не удаляй их, добавь `"notion"` в существующий объект `mcpServers`
- **Невалидный JSON**: Если файл повреждён — перезапиши, но предупреди пользователя
