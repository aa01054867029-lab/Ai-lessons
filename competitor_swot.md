# SWOT-анализ конкурентов

> В контексте проектирования Compliance-платформы для финтех-стартапов РФ (115-ФЗ)

---

## 1. Sumsub

| | |
|---|---|
| **Strengths (Конкурент)** | **Weaknesses (Конкурент)** |
| ✅ All-in-one: KYC + AML + fraud в одном продукте. 14 000+ типов документов, 220+ стран | ❌ Перегруженность: стартапы жалуются на крутую кривую обучения и пробелы в документации |
| ✅ No-code workflow builder с визуальными branches | ❌ Скрытые микротранзакции: +$0.35 за liveness, +$0.08 за AML-матч — бюджет непредсказуем |
| ✅ Верификация за <60 сек, мобильный SDK | ❌ Нет специфики 115-ФЗ: отсутствует проверка по Росфинмониторингу, формы ФЭС 1-ФМ/2-ФМ |
| ✅ AI-копилот «Summy» для автоматизации разбора | ❌ Нет approval workflow для разрешения конфликтов product ↔ compliance |
| **Opportunities (Для нас)** | **Threats (Для нас)** |
| 🟢 Занять нишу «Sumsub для РФ» с 115-ФЗ из коробки | 🔴 Sumsub может добавить российскую специфику — глобальный бренд + ресурсы |
| 🟢 Предложить прозрачное ценообразование vs. скрытые надбавки | 🔴 Сильная экосистема SDK и интеграций — трудно конкурировать по охвату |
| 🟢 Упрощённый onboarding для маленьких команд без compliance-инженеров | 🔴 Большая пользовательская база = социальное доказательство |

## 2. ComplyAdvantage

| | |
|---|---|
| **Strengths (Конкурент)** | **Weaknesses (Конкурент)** |
| ✅ Эталонное качество risk intelligence: AI + NLP, обновление в реальном времени | ❌ **Не делает KYC** — нет верификации документов, нужен второй инструмент |
| ✅ ComplyLaunch — бесплатный enterprise-tier для ранних стартапов (vendor lock-in) | ❌ Модуль мониторинга транзакций негибкий для кастомных поведенческих паттернов |
| ✅ No-code rule builder, доступный не-техническим пользователям | ❌ Скрытые допзатраты: re-screening, места поддержки, долгосрочное хранение |
| ✅ Starter план от $99/мес за 1000 сущностей | ❌ Нет поддержки перечня Росфинмониторинга и ГОСТ-криптографии для ФЭС |
| **Opportunities (Для нас)** | **Threats (Для нас)** |
| 🟢 Единая платформа KYC + AML (vs. 2 инструмента у пользователей CA) | 🔴 ComplyLaunch — мощный инструмент захвата: бесплатно → lock-in → платно |
| 🟢 Скопировать модель ComplyLaunch для российского рынка | 🔴 Лучшие данные по глобальным санкциям и PEP — сложно конкурировать по полноте |
| 🟢 Гибкий мониторинг транзакций с IDF-взвешиванием для кириллицы | 🔴 Сильный бренд в enterprise-сегменте |

## 3. Unit21

| | |
|---|---|
| **Strengths (Конкурент)** | **Weaknesses (Конкурент)** |
| ✅ Лучший UX для compliance-офицеров: alert queue, приоритизация, кастомные workflow | ❌ «Спагетти-код» из правил: при масштабировании no-code правила начинают конфликтовать |
| ✅ Shadow mode + бэктестинг: тест новых правил на исторических данных | ❌ Заградительная цена: средний контракт ~$160K/год, минимум от $33K |
| ✅ Agentic AI: ИИ-агенты сокращают разбор алертов с 14ч до 41мин/день | ❌ Фокус на US/EU — нет поддержки РФ-юрисдикции |
| ✅ Латентность <250ms для realtime-решений | ❌ Нет KYC-верификации (документы, биометрия) |
| **Opportunities (Для нас)** | **Threats (Для нас)** |
| 🟢 Перенять паттерны UX alert queue, но с анти-спагетти ИИ-ассистентом | 🔴 Agentic AI — технологический авангард, сложно догнать по ML |
| 🟢 Доступная цена для стартапов vs. $160K/год | 🔴 Может выйти на российский рынок через партнёрства |
| 🟢 Единая платформа KYC + AML + Approval (vs. только AML у Unit21) | 🔴 Сильные enterprise-клиенты = кейсы и рефералы |

## 4. Salv

| | |
|---|---|
| **Strengths (Конкурент)** | **Weaknesses (Конкурент)** |
| ✅ Salv Bridge: уникальный collaborative intelligence — обмен данными между банками (GDPR) | ❌ Фокус на EU/Nordics — нет присутствия в РФ |
| ✅ Полный AML-стек: мониторинг, скрининг, risk scoring, case management | ❌ Custom pricing — недоступно для оценки, нет starter tiers |
| ✅ Сильный акцент на recovery fraud-средств (APP fraud) | ❌ Нет KYC/верификации документов |
| ✅ Кастомизируемый rules engine | ❌ Нет no-code/low-code builder — нужны инженерные ресурсы |
| **Opportunities (Для нас)** | **Threats (Для нас)** |
| 🟢 Концепция collaborative intelligence — заложить в roadmap (обмен данными между финтехами РФ) | 🔴 Минимальная — Salv не присутствует на российском рынке |
| 🟢 Предложить self-service onboarding vs. enterprise-only модель | |

## 5. Ondato

| | |
|---|---|
| **Strengths (Конкурент)** | **Weaknesses (Конкурент)** |
| ✅ Скорость верификации: KYC за 30-60 сек, точность 99.8% | ❌ Базовый case management — не подходит для 50-100 алертов/день |
| ✅ 192 страны, сертификация ISO 27001, eIDAS | ❌ Дорого для маленьких стартапов |
| ✅ Чистый, интуитивный UI — хвалят администраторы | ❌ Интеграция с legacy-системами бывает сложной |
| ✅ Биометрия + liveness + возрастная верификация (AI-сертифицированная) | ❌ Нет AML-мониторинга транзакций — только screening |
| **Opportunities (Для нас)** | **Threats (Для нас)** |
| 🟢 Ориентир для UX: чистый UI + быстрая верификация | 🔴 Может выйти в РФ через eIDAS-совместимость |
| 🟢 Полный стек KYC + AML в одном продукте vs. только KYC | 🔴 Высокая точность верификации (99.8%) — планка для нашего ML |
