# Screens Index — ComplyFlow

## Основной поток CO (финальные экраны, Figma → Screens section)

- `dashboard-co.md` — рабочий стол CO: StatCard × 3, превью алертов, QuickActions, RecentActivity
- `alert-queue.md` — очередь алертов: фильтры, таблица × 8 строк, Pagination, BulkActions
- `alert-card.md` — карточка алерта: детали транзакции, AML-флаги, риск-скор, история, комментарий
- `approval-request-card.md` — запрос на согласование: детали + обсуждение (CommentItem × 3) + DecisionCard
- `escalation-card.md` — эскалация (mobile 375px): AlertBanner, карточки сути/рекомендации/данных, footer с кнопками решения

## Ранние наброски (wireframe stubs)

- `dashboard.md` — главный экран: KPI-метрики, таблица клиентов, активность
- `clients.md` — список клиентов с поиском, фильтрами, таблицей, пагинацией
- `documents.md` — реестр документов с фильтрами по типу и статусу
- `risks.md` — риск-оценки: сводные карточки по уровням + таблица оценок
- `settings.md` — настройки профиля организации (форма + checkbox)
