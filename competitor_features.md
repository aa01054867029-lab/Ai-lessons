## 🏆 Уникальные фичи — есть у нас, нет ни у одного конкурента

| Фича | Почему это важно |
|---|---|
| **Approval workflow (Maker-Checker) с эскалацией CEO** | Единственный инструмент разрешения конфликта product ↔ compliance. Золотой стандарт (Four Eyes Principle) из банковской практики, но отсутствует у всех SaaS-конкурентов |
| **Перечень Росфинмониторинга + ФЭС 1-ФМ/2-ФМ** | Ни один глобальный конкурент не поддерживает российскую отчётность. Стартапы вынуждены строить SOAP-шлюз с ГОСТ-криптографией самостоятельно |
| **ГОСТ-криптография (УКЭП) для отчётности в РФМ** | Архитектурный барьер: REST → SOAP + WS-Security. Наша платформа = BFF-прокси, принимающий JSON и отдающий зашифрованный XML |
| **Policy management с версионированием** | Обязательно для аудитов ЦБ — кто, когда и почему изменил правила. Конкуренты не предоставляют |
| **IDF-взвешивание для кириллицы** | Снижение false positive с 90% до <15% при скрининге русских/арабских имён — главная боль аналитиков |
| **CEO Dashboard с override-логом** | Топ-менеджер видит картину рисков и принимает решения — нет ни у одного конкурента |
| **BNPL-модуль (283-ФЗ + БКИ)** | Новый закон (апрель 2026) создал рынок ~940 млрд руб., требующий compliance. Захват аудитории |
