# Ai-lessons

React + Vite component library and Storybook catalog for ComplyFlow UI kit.

## Что в проекте
- `src/components/` — React design system components
- `src/tokens/` — CSS token definitions (primitive, semantic, typography)
- `src/stories/` — Storybook catalog для компонентов и foundation
- `.storybook/` — Storybook конфигурация

## Запуск
```bash
npm install
npm run dev
```

## Storybook
```bash
npm run storybook
```
